# Карта для НРИ

Интерактивная карта для настольной ролевой игры: мастер загружает изображение карты, карта разбивается на клетки, игроки открывают клетки по мере прохождения, а мастер может отмечать точки, где нужны проверки (скрытность, восприятие и т.д.).

## Как пользоваться

1. **Загрузить карту** — нажмите «Загрузить карту» и выберите изображение (PNG, JPG и т.д.).
2. **Сетка** — задайте число клеток по горизонтали и вертикали (по умолчанию 8×6). Карта автоматически разобьётся на клетки.
3. **Открытие клеток** — включите режим «Открыть клетки», выберите «1 клетку», «3×3» или «5×5», и кликайте по затемнённым клеткам. Они будут открываться по мере продвижения игроков.
4. **Точки проверок** — нажмите «Добавить проверку», затем кликните по карте в нужном месте. Введите название и тип проверки (например: «Засада гоблинов», «Скрытность»). Точки отображаются номерами на карте и в списке слева. Удалить точку можно кликом по маркеру на карте или по кнопке «×» в списке.
5. **Сброс** — «Сбросить карту» удаляет изображение, все открытые клетки и точки. Состояние по умолчанию сохраняется в браузере (localStorage).

## Запуск

Откройте файл `index.html` в браузере (двойной клик или «Открыть с помощью» → браузер). Сервер не нужен.

## Технологии

HTML, CSS, JavaScript. Данные хранятся в `localStorage` в том же браузере.
